Overview

This project demonstrates a very simple Shiny application implemented using R.

User Guidance

The application allows the user to select an automotive feature from a list. The application then creates a scatter plot showing the selected feature on the X axis against the mpg feature on the Y axis. Also, a linear regresion line is added to the plot showing the X/Y regression.

Slide with R Output - Automotive feature list

The following automotive features are presented to the user for selection:

##  [1] "cyl"  "disp" "hp"   "drat" "wt"   "qsec" "vs"   "am"   "gear" "carb"
